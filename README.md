2.排序基础
----------

>>普及一下排序稳定，所谓排序稳定就是指：如果两个数相同，对他们进行的排序结果为他们的相对顺序不变。（由代码决定，而与算法
>>无关）

>>普及一下原地排序：原地排序就是指不申请多余的空间来进行的排序，就是在原来的排序数据中比较和交换的排序。例如快速排序，堆
>>排序等都是原地排序，合并排序，计数排序等不是原地排序。
